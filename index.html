CSS Snippet Cheatsheet

Ever have trouble recalling your favorite CSS code?

Flexbox Row

Use these properties to create a Flexbox row layout

body{
        background:salmon;
  }

Flex Center

Get your items to show up right in the middle with flexbox centering

.container{
        display:flex;
        justify-content:center;
        align-items:center;
        }

Glow Shadow

Get the cool shadow effect of these cards

.card:hover, .card:hover h3 {
    box-shadow: 0 0 10px inset chartreuse, 
    0 0 10px deepskyblue;
        }

Two-toned background

Sweet two toned effect for a 90s feel

body {
background:linear-gradient(
    120deg,
    salmon 0%,
    salmon 50%
    deepskyblue 50%
    );
    }
